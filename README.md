# Air Draw

Draw in the air using hand gestures tracked by your webcam.

## Features

- **Pinch Mode**: Pinch your thumb and index finger together to draw
- **Index Mode**: Extend your index finger while keeping other fingers down to draw
- Real-time hand tracking powered by [MediaPipe](https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker)
- Adjustable brush size, smoothing, and opacity
- Eraser mode
- Save your drawings as PNG images

## Usage

1. Allow camera access when prompted
2. Choose your preferred mode (Pinch or Index)
3. Start drawing in the air!
4. Adjust controls in the bottom-right panel
5. Click "Save" to download your artwork

## Technology

Built with:
- [MediaPipe Hands](https://ai.google.dev/edge/mediapipe/solutions/vision/hand_landmarker) for real-time hand tracking
- HTML5 Canvas for rendering
- Vanilla JavaScript
